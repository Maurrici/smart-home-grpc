# smart-home-grpc
Este projeto é dividido em duas aplicações:
 - client_proxy: Servidor Node com comunicação gRPC para o servidor [Servidor Python](https://github.com/jvtesteves/smart_home_parte_2) e Websocket para transmitir as informações para os clientes web conectados
 - client_interface: Aplicação web que prover uma interface de comunicação com o servidor Node para usuários
